## 🚨 常见问题快速排查

### MessageList 不显示
1. 检查控制台错误
2. 确认 `chatStore.messages` 有数据
3. 确认组件正确导入
4. 检查网络请求是否成功

### 显示 "Unknown User"
1. 检查后端返回的数据格式
2. 确认 `normalizeMessage` 函数工作正常
3. 检查 sender_id/sender_name 字段

### API 请求失败
1. 检查后端是否运行
2. 确认 API URL 配置正确
3. 检查跨域设置
4. 验证 Token 是否有效
