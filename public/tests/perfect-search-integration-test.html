🎯 测试步骤

  1. 打开聊天页面: 点击"打开聊天页面"按钮或访问 /chat/{chatId}
  2. 点击搜索图标: 在聊天页面 header 右侧点击 "Search" 按钮
  3. 使用快捷键: 按 Cmd+K (Mac) 或 Ctrl+K (Windows)
  4. 输入搜索内容: 在弹出的搜索框中输入关键词
  5. 选择搜索结果: 点击结果项目进行导航
  6. 验证导航: 确认页面跳转到正确的消息位置
