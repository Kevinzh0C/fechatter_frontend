üìã How to Use URL Direct Login

You can log in directly by adding a ?user= parameter to any URL:

https://your-app.vercel.app/?user=admin https://your-app.vercel.app/chat/1?user=user1 https://your-app.vercel.app/settings?user=demo

The system will automatically log in the specified user and redirect to the intended page.
